### v1.2 - Vue 框架支持

#### 3. Vue 组件提取 ✅
- **新增提取器**: `src/extractors/vue-component-extractor.ts`
- **功能**: 从 `.vue` 文件提取组件信息
- **测试**: 100% 覆盖

**支持的 Vue 特性**:
- Vue 3 Composition API (`<script setup>`)
- Vue 2 Options API (`export default`)
- Props 提取（TypeScript 和 Runtime）
- Events 提取（`defineEmits`, `emit()` 调用）
- Slots 提取（`<slot>` 标签）

**示例 - Vue 3 Composition API**:
```vue
<template>
  <button @click="handleClick">
    <slot />
  </button>
</template>

<script setup lang="ts">
defineProps<{
  variant: 'primary' | 'secondary';
  size?: 'sm' | 'lg';
}>();

const emit = defineEmits<{
  click: [];
  change: [value: string];
}>();
</script>
```

提取结果：
- Props: `variant` (required), `size` (optional)
- Events: `click`, `change`
- Slots: `default`
- API Style: `Composition`

**示例 - Vue 2 Options API**:
```vue
<script>
export default {
  name: 'Button',
  props: {
    variant: {
      type: String,
      required: true
    },
    size: {
      type: String,
      default: 'md'
    }
  }
}
</script>
```

#### 4. Vue 组件内样式提取 ✅
- **新增提取器**: `src/extractors/vue-style-extractor.ts`
- **功能**: 从 Vue SFC 的 `<style>` 标签提取样式
- **测试**: 100% 覆盖

**支持的样式语言**:
- CSS (`<style>`)
- Less (`<style lang="less">`)
- SCSS (`<style lang="scss">`)
- Scoped 样式识别

**示例**:
```vue
<template>
  <div class="container">Content</div>
</template>

<style lang="less" scoped>
@primary: #3b82f6;

.container {
  background: @primary;
  padding: 1rem;
}
</style>
```

提取结果：
- Variables: `@primary` (Less Variable)
- Utilities: `.container` (scoped)
- Lang: `less`

## 📝 使用示例

### 1. 提取 Less 样式

```typescript
import { extractLess } from 'ai-codegen-booster';

const lessContent = `
  @primary-color: #3b82f6;
  .button { background: @primary-color; }
`;

const result = await extractLess(lessContent);
console.log(result.variables); // [{ name: '@primary-color', value: '#3b82f6', category: 'Color' }]
```

### 2. 提取 Vue 组件

```typescript
import { extractVueComponent } from 'ai-codegen-booster';

const vueCode = `
<template>
  <button @click="handleClick">
    <slot />
  </button>
</template>

<script setup lang="ts">
defineProps<{
  variant: 'primary' | 'secondary';
}>();

const emit = defineEmits<{ click: [] }>();
</script>
`;

const result = await extractVueComponent(vueCode);
console.log(result.framework); // 'Vue'
console.log(result.props);     // [{ name: 'variant', type: "'primary' | 'secondary'", required: true }]
console.log(result.events);    // ['click']
```

### 3. 提取 Vue 组件内样式

```typescript
import { extractVueStyles } from 'ai-codegen-booster';

const vueCode = `
<template><div /></template>

<style lang="less" scoped>
@primary: #3b82f6;
.container { background: @primary; }
</style>
`;

const result = await extractVueStyles(vueCode);
console.log(result.styles[0].lang);    // 'less'
console.log(result.styles[0].scoped);  // true
console.log(result.variables);         // [{ name: '@primary', value: '#3b82f6', lang: 'less' }]
```
