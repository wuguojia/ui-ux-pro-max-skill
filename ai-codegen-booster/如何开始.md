# AI Codegen Booster - 如何开始使用

> 3分钟快速集成，让AI更懂你的项目

**版本**: v1.4.0
**适用于**: Claude Code, Cursor, Windsurf, Codeium 等AI编程助手

---

## 📖 这是什么？

AI Codegen Booster 是一个 **Skill（技能）**，而不是传统的命令行工具。它让 AI 编程助手能够：

- ✅ 自动提取你项目中的组件、样式、编码规范
- ✅ 理解你的项目架构和设计模式
- ✅ 生成符合你项目规范的代码
- ✅ 智能去重和质量评分

---

## 🚀 三种使用方式

### 方式 1：直接对话（最简单）⭐ 推荐

**适用场景**：使用 Claude Code, Cursor 等支持对话的 AI 编程助手

1. **打开你的项目**
   ```bash
   cd your-project
   ```

2. **直接对 AI 说**：
   ```
   "帮我分析这个 Vue/React 项目，提取所有组件、样式和编码规范到知识库"
   ```

3. **AI 会自动**：
   - 扫描项目文件
   - 提取组件信息（Props、事件、Hooks、Composables等）
   - 分析样式系统（CSS变量、工具类）
   - 检测设计模式和架构模式
   - 提取编码约定和最佳实践
   - 智能去重和质量评分
   - 生成知识库 CSV 文件

4. **开始使用**：
   ```
   "用我们项目的组件创建一个登录表单"
   ```

AI 会使用刚才提取的知识库，生成完全符合你项目规范的代码！

---

### 方式 2：作为 Skill 集成到 Claude Code

**适用场景**：想要长期使用，每次打开项目自动加载

#### 步骤 1: 复制 Skill 到你的项目

```bash
# 在你的项目根目录
mkdir -p .claude/skills
cp -r path/to/ai-codegen-booster .claude/skills/
```

#### 步骤 2: 创建 Skill 配置

在项目根目录创建 `.claude/skills/ai-codegen-booster/SKILL.md`（如果还没有的话）

#### 步骤 3: 重启 Claude Code

关闭并重新打开 Claude Code，它会自动加载这个 Skill。

#### 步骤 4: 验证 Skill 已加载

对 AI 说：
```
"你有 ai-codegen-booster 这个技能吗？"
```

AI 应该回答它有这个技能，并能描述功能。

---

### 方式 3：使用命令行工具（备选）

**适用场景**：需要脚本化、自动化的场景

```bash
# 进入工具目录
cd ai-codegen-booster

# 安装依赖
npm install

# 运行搜索
npm run search "button" --domain components
```

> ⚠️ 注意：命令行方式需要手动配置，不如 AI 辅助方式智能，仅作为备选方案。

## 🎯 核心功能

### 1. 智能提取器

- **HTML 提取器** - 提取 HTML 结构模式、语义化标签、ARIA 可访问性
- **文档提取器** - 提取 JSDoc/TSDoc/Markdown 文档
- **React Hooks 提取器** - 提取 useState、useEffect、自定义 Hook、Context API
- **Vue Composables 提取器** - 提取 ref、reactive、composables、指令

### 2. 智能分析器

- **模式分析器** - 识别设计模式（单例、工厂、观察者、构建器）
- **架构分析器** - 分析项目架构（MVC、MVVM、Clean Architecture）
- **约定分析器** - 自动提取命名约定、检测反模式

### 3. 质量优化

- **智能去重器** - 基于相似度算法合并重复组件
- **质量评分器** - 0-100 分评分系统，评估完整性、一致性、可复用性

## 📁 生成的知识库结构

运行后会在项目中生成 `kb/` 目录：

```
your-project/
├── kb/                          # 知识库目录
│   ├── components.csv           # 组件知识库
│   ├── styles.csv              # 样式知识库
│   ├── conventions.csv         # 编码规范知识库
│   └── quality-report.md       # 质量报告
├── src/
│   ├── components/
│   └── styles/
└── .claude/
    └── skills/
        └── ai-codegen-booster/  # Skill 目录
```

## ❓ 常见问题

### Q1: 这个 Skill 支持哪些框架？

**A**: 支持主流前端框架：
- ✅ React (包括 Next.js)
- ✅ Vue (2.x 和 3.x)
- ✅ Svelte
- ✅ Angular
- ✅ 纯 HTML/CSS/JavaScript

### Q2: 需要手动编写 CSV 文件吗？

**A**: **不需要！** AI 会自动提取和生成。你只需要：
1. 告诉 AI 分析你的项目
2. AI 自动生成 CSV
3. 开始使用

### Q3: 会修改我的项目代码吗？

**A**: **不会。** 这个 Skill 只会：
- 读取你的代码文件
- 分析和提取信息
- 生成知识库 CSV 文件（在 `kb/` 目录）
- 不会修改任何源代码

### Q4: 多人协作怎么办？

**A**: 建议：
1. 将 `kb/` 目录提交到 Git
2. 团队成员共享同一个知识库
3. 有新组件时，让 AI 更新知识库
4. 定期运行质量评分，保持知识库质量

### Q5: 知识库需要更新吗？

**A**: 当你的项目有新组件或样式时，对 AI 说：
```
"更新知识库，扫描新增的组件"
```

AI 会：
- 增量扫描（只扫描变化的文件）
- 自动去重
- 更新 CSV 文件
