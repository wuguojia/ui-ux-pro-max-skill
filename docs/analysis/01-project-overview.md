# 01 — 项目总览与定位

## 一句话概括

UI/UX Pro Max 是一个 **AI 驱动的设计智能工具包**，以 CSV 数据库 + BM25 搜索引擎的形式，为 18 个 AI 编码助手平台提供专业的 UI/UX 设计知识。

---

## 它解决了什么问题？

### 问题现状

当开发者使用 AI 编码助手（Claude Code、Cursor、Copilot 等）来构建 UI 时，AI 缺乏：

1. **系统化的设计知识** — 不知道 SaaS 产品应该用什么风格、什么配色
2. **行业特定的规则** — 金融 App 和游戏 App 的设计原则截然不同
3. **可检索的设计数据** — 需要从海量设计知识中快速定位相关信息
4. **一致的设计系统** — 从风格、配色、字体到组件的完整推荐

### 解决方案

UI/UX Pro Max 通过以下方式解决这些问题：

```
用户: "帮我做一个 SaaS Dashboard"
                ↓
AI 调用: python3 search.py "SaaS dashboard" --design-system
                ↓
搜索引擎在 42 个 CSV 中搜索 → 推理引擎匹配 161 条规则
                ↓
输出: 完整设计系统 (风格: Glassmorphism + Flat, 配色: Trust Blue,
      字体: Space Grotesk/DM Sans, 布局: Hero+Features+CTA, 反模式: ...)
                ↓
AI 根据设计系统生成高质量 UI 代码
```

## 技术栈

| 层 | 技术 | 为什么选它 |
|----|------|-----------|
| 搜索引擎 | Python 3（无外部依赖） | 任何环境都能跑，零安装成本 |
| BM25 算法 | 纯 Python 实现 | 不需要向量数据库或 embedding |
| 数据格式 | CSV | 人可读、Git 友好、Excel 可编辑 |
| CLI 安装器 | TypeScript + Bun | 类型安全、打包快 |
| 包管理 | npm (uipro-cli) | 全球开发者最熟悉的包管理器 |
| 模板引擎 | 自研 (`{{PLACEHOLDER}}`) | 零依赖、足够简单 |

## 用户旅程

```
┌─────────────────────────────────────────────────────────┐
│  Step 1: 安装                                            │
│  $ npx uipro-cli init --ai claude                       │
│  → 自动在 .claude/skills/ui-ux-pro-max/ 创建文件         │
├─────────────────────────────────────────────────────────┤
│  Step 2: AI 自动调用                                     │
│  用户: "做一个电商首页"                                    │
│  AI 读取 SKILL.md → 调用 search.py → 获取设计系统         │
├─────────────────────────────────────────────────────────┤
│  Step 3: 按推荐实施                                      │
│  AI 根据推荐的风格、配色、字体、布局生成代码               │
│  用户对代码进行微调                                       │
├─────────────────────────────────────────────────────────┤
│  Step 4: 持久化（可选）                                   │
│  --persist 保存设计系统到 design-system/MASTER.md         │
│  后续页面自动复用同一套设计规则                             │
└─────────────────────────────────────────────────────────┘
```

## 项目定位与竞品对比

| 特性 | UI/UX Pro Max | Figma | Tailwind UI | Design Tokens |
|------|--------------|-------|-------------|---------------|
| AI 原生集成 | ✅ 18 个平台 | ❌ | ❌ | ❌ |
| 行业推理 | ✅ 161 规则 | ❌ | ❌ | ❌ |
| 零依赖 | ✅ 纯 Python | ❌ 需要客户端 | ❌ 需要 Tailwind | ❌ 需要工具链 |
| 可搜索 | ✅ BM25 | ❌ 手动浏览 | ❌ 手动浏览 | ❌ |
| 开源 | ✅ MIT | 部分 | 付费 | 开源 |
| 数据可编辑 | ✅ CSV | ❌ 专有格式 | ❌ | ✅ JSON |

**核心差异化**: UI/UX Pro Max 是唯一一个 **让 AI 编码助手理解设计规则** 的工具。它不是给人看的组件库，而是给 AI 读的设计知识库。
