# 04 — 未来路线图与演进方向

## 短期 (1-3 个月)

### 1. TypeScript 完整重写

当前状态: `core.ts` 已完成 (680 行)，`design_system.ts` 和 `search.ts` 待写。

目标:
- 搜索引擎 Python/TypeScript 双版本
- CLI 可直接调用 TS 版本，无需 Python
- 对 Node.js/Bun 用户更友好

### 2. 自动化测试

- [ ] BM25 搜索准确性测试 (查询-期望结果对)
- [ ] CSV 数据一致性检查
- [ ] CLI 安装/卸载 E2E 测试
- [ ] 跨平台生成文件验证

### 3. CI/CD 流水线

```yaml
# 目标:
on push:
  - 运行搜索测试
  - 验证 CSV 格式
  - 构建 CLI
on tag:
  - 发布 npm 包
  - 创建 GitHub Release
```

## 中期 (3-6 个月)

### 4. Skill 模板化框架

将当前架构抽象为通用框架:
- `create-skill` 脚手架工具
- 配置驱动的域注册
- 模板市场

### 5. 向量搜索增强

在 BM25 基础上可选添加语义搜索:
- 预计算 CSV 行的 embedding
- BM25 + 余弦相似度混合排名
- 对"模糊"查询更好 (如"让用户感觉信任的设计")

### 6. 实时数据更新

- `uipro update --data-only` 只更新 CSV 数据
- 增量更新而非全量替换
- 版本化数据 (data v1, v2...)

---

## 长期 (6-12 个月)

### 7. 交互式设计系统编辑器

Web UI 用于:
- 可视化浏览 67 种风格
- 实时预览 161 种配色
- 拖拽组合设计系统
- 导出为 MASTER.md

### 8. 社区驱动的数据

- CSV 贡献指南
- PR 自动验证 (格式检查 + 测试)
- 社区投票排名
- 领域专家审核

### 9. 多语言支持

- 中文/日文/韩文搜索
- Unicode 分词
- 多语言 Skill 模板
